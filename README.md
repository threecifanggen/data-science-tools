## EXCEL

### 1. 二项分布检验

**依赖**：Office Excel >= 2015

**功能**：用贝叶斯/Bootstrap+参数检验验证两个二项分布的分数(留存率、转化率等)是否有显著性差异。

**文件地址**: [excel/二项分布检验.xlsx](excel/二项分布检验.xlsx)

#### 使用方法

只需要在`L7`、`L8`、`M7`、`M8`中填入对应数字即可。
